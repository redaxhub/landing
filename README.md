
# REDAX Hub Landing

Vite React landing page for REDAX Hub.

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Build output directory: `dist`

## Vercel Settings

- Framework preset: `Vite`
- Build command: `npm run build`
- Output directory: `dist`
